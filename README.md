## 🧭 What is StudyHub?

StudyHub is a centralized academic platform designed to eliminate chaos in student learning.

Instead of juggling multiple sources, StudyHub provides a unified environment where students can access notes, previous year question papers, and structured resources instantly.

It is built with one goal:

> **Make studying faster, cleaner, and more effective.**
