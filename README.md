## Overview

StudyHub is a mobile platform built to make academic resources feel simple, fast, and accessible.

It brings together:

- course-wise notes
- previous year question papers
- personal dashboard access
- profile management
- dark mode support
- cloud-based storage

The goal is to reduce friction in exam preparation and give students a focused, organized place to study.
