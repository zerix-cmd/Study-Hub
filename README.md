## 🧭 Overview

StudyHub is a centralized learning platform that helps university students access and organize academic resources in one place.

Instead of relying on scattered materials, StudyHub provides a structured system where notes, previous year question papers, and academic content are easily accessible.

The goal is simple:

> **Spend less time searching. Spend more time learning.**

---

## ⚙️ Core Features

### 📚 1. Structured Notes System
StudyHub organizes notes based on real academic structure:

- Category-based filtering (Major / Minor / AEC / MDC / VAC)  
- Subject-wise organization  
- Semester and module-level breakdown  
- Search-based access for quick retrieval  

This ensures that students can navigate content logically instead of manually searching through files.

---

### 📄 2. Previous Year Question Papers (PYQs)
StudyHub includes a dedicated system for exam preparation:

- Subject-wise PYQ access  
- Organized by semester  
- Helps identify exam patterns  
- Enables faster revision before exams  

---

### 🔍 3. Advanced Search System
The search functionality is designed to be fast and practical:

- Keyword-based search  
- Instant filtering  
- Works across notes and PYQs  
- Reduces navigation time significantly  

---

### 🧠 4. Smart Dashboard
The dashboard acts as the central control system:

- Quick access to recent materials  
- Organized layout for better usability  
- Minimal UI for distraction-free experience  

---

### 🤖 5. AI-Powered Assistance
StudyHub integrates AI to enhance learning:

- Summarize notes instantly  
- Generate key points  
- Create exam questions and MCQs  
- Assist in understanding complex topics  

---

### 📂 6. File Upload & Management
Users can contribute and manage content:

- Upload PDF notes  
- Organized storage system  
- Admin/user control over content  
- Secure cloud-based handling  

---

### 👤 7. Profile & Personalization
Each user gets a customized experience:

- Profile management  
- Preference-based filtering (University / Program)  
- Role-based access (User / Admin)  

---

### 🌙 8. Dark Mode Experience
Designed for long usage:

- Reduced eye strain  
- Smooth UI consistency  
- Persistent theme settings  

---

### ⚡ 9. High Performance Experience
Optimized for smooth usage:

- Fast loading screens  
- Smooth scrolling  
- Optimized PDF viewer  
- Efficient backend queries  

---

### ☁️ 10. Cloud-Based Architecture
Powered by modern backend systems:

- Supabase (Auth + Database + Storage)  
- PostgreSQL database  
- Secure and scalable
