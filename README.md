# Study Hub 📚

**Study Hub** is a mobile application designed to help university students access study materials, notes, and previous year question papers in one place. The app aims to simplify exam preparation by providing organized academic resources through a clean and user-friendly interface.

## Features
- Student login and authentication  
- Course-wise study notes  
- Previous year question papers  
- Personal dashboard  
- Profile management  
- Dark mode support  
- Cloud-based data storage  

## Technology Stack
- **Frontend:** Flutter  
- **Backend:** Supabase  
- **Database:** PostgreSQL  
- **UI:** Material Design  
- **Development:** AI-assisted tools  

## Vision
Study Hub aims to create a centralized platform where students can easily access learning resources, stay organized, and prepare effectively for university exams.
